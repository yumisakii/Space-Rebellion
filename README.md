# Space Rebellion (C++ / SFML)

Space Rebellion is a 2D horizontal shoot 'em up, created as a student project at Gaming Campus (G.Tech). It was built from scratch in C++ using the SFML multimedia library.

## üåü Key Features

* **Dynamic Gameplay:** Navigate your ship and shoot your way through waves of enemies.
* **Weapon System:** Wield standard projectiles and powerful special missiles.
* **Enemy Variety:** Battle 3 distinct enemy types with unique behaviors, plus a final Boss.
* **Destructible Obstacles:** Destroy environmental hazards to gain bonuses.
* **Main Menu:** Includes game settings, such as a volume/sound controller.

## üõ†Ô∏è How to Run (for Developers)

This project was built using **Visual Studio 2022**.

1.  Clone the repository to your local machine.
2.  Open the solution file (`.sln`) in Visual Studio 2022.
3.  Set the solution configuration to **Release** and the platform to **x64**.
4.  Build and run the project once from Visual Studio.
5.  After the initial build, you can run the game directly from the `x64/Release/Game.exe` file.
